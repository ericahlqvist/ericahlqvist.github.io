My research has mainly been focused on

  • the multiplicative structure of the étale cohomology of number fields and applications in number theory -- class field towers, Galois representations, etc.
  • building data for certain algebraic stacks, stacky modifications, root stacks, Artin--Schreier stacks, compactification
  • good moduli spaces in derived algebraic geometry.

My papers and preprints can be found on the ArXiv.

Some C code for computing Massey products in the étale cohomology of rings of integers of imaginary quadratic fields using PARI can be found on my GitHub.
